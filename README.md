# AI Lead Automation System

This project demonstrates an AI-based lead automation workflow built using Zapier.

## Workflow

Google Form → Google Sheets → AI Analysis → Processed Leads Sheet → Automated Email

## Features

- Automatic lead capture
- AI-based message analysis
- Lead categorization
- Auto-reply generation
- Email automation

## Tools Used

- Zapier
- Google Sheets
- AI (GPT-based)
- Email Automation

## Note

Zapier configuration and client data are not included for security reasons.
